== Description ==


Bigboss slider for creating the Modern slideshow in any area of your WordPress sites like in post page or sidebar area. responsive  Custom post type slider, easy to use, install and customize.

used this shortcode " [bigboss_slider] " in page or post 
or used this " < ?php bigboss_slider() ; ? >  "   php code in template .

== Installation ==
How To install Bigboss slider plugin on your wordpress site 
1.Upload plugin-name.php to the /wp-content/plugins/ directory

2.Activate the plugin through the 'Plugins' menu in WordPress 

3.use this code " < ?php bigboss_slider() ; ? > " to show up slider 
  plugin in your template .

4.or your can used this shortcode " [bigboss_slider] " to add slider on page/post
